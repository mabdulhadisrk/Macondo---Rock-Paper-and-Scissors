# Macondo---Rock-Paper-and-Scissors
This is my second Macondo software project built using HTML, CSS, and JavaScript. The core game is based on classic "Rock, Paper, Scissors." However, I wanted to make it more entertaining and catchy, so I changed the theme to cosmic space. I renamed the options and Name to Atom, Hole, and Cyclone. I chose this theme because I love learning about space, galaxies, and physics. Because JavaScript is often the hardest part of a game due to its complex syntax and variable naming, I used Gemini to help me check and fix my code errors.
# Languages
HTML: Used to build the structure of the game.
CSS: Used for styling the interface and adding the space background image.
JavaScript: Used to handle the game's logic and functionality.
# Reason of Making
As a beginner, I was looking for simple yet fun projects. I decided to make a "Rock, Paper, Scissors" variation because it is a globally recognized game played by people of all ages, making it both fun to build and play.
# Features
Custom Theme: Features beautiful, custom icons that replace standard rock, paper, and scissors with space elements.
Score Tracker: Automatically tracks and displays your wins, losses, and draws in real-time.
Smart Bot Logic: Includes custom text alerts that change based on whether you win, lose, or tie with the bot.
